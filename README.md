# Stopots bot

## 👀 Overview

This is a application made using puppeteer to create a bot that plays [stopots](https://stopots.com/en/) game alone.

![Bot](.github/bot-stopots.gif)

## ⚙️ Installation

```
  # git clone https://github.com/RaphaelOliveiraMoura/stopots-bot.git
  # yarn install (or npm)
  # yarn start
```
